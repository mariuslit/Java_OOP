Basic Input Output System - BIOS

BIOS dirba tik su MBR
pagrindinis stabdis buvo kad reikalavo kad turetų MBR, o MBR negalejo nuskaityti įrenginių, didesnių už 2TB.

MBR (Master Boot Record) turi 12 baitų kodą skirtą įkrauti operacinę sistemą

UEFI BIOS (BIOS'o išplėtimas) Unified Extensible Firmware Interface, naujas BIOS pakaitalas                  
nuskaito iki 22TB (2ZB)
GPT - naudojamas vietoje MBR
GPT - diskų skaidymas dalimis yra skirtingi, jeigu bIOS nepalaiko GPT nepavyks perkelti sistemos iš vieno kompiuteio į kitą

BIOS'o paleidimas užkrovimo metu: F1, F2, F10, Delete (KURI MANO ??????????????????)

BIOS meniu: | File | Storage | Security | Power | Advanced |

BIOS'e galima
Power >> OS Power Managment - galima išjungti, nes tai skirta elektros taupymui
Storage >> Boot Order - iš kurių įrenginių krauti OS, kai keičiame diską (HDD, SSD)
File >> Set Time and Date - galima pasikeisti datą ir laiką kai keičiama baterija.

Operacinė Sistema - OS

OS paskirtis
1 suteikti grafinę vartotojo sąsają - matomiausia
2 Valdyti išorinius įrenginius (printeriai, klaviatūra, monitorius)
3 Atminčių valdymas (pastovioji, išorinė, RAM, laikmenos) - nematoma bet svarbiausia nuo ko priklauso kompiuterio greitis
4 API (Application Programming Interface) - pateikti, OS teikia paslaugą  programuotojui (API)
5 Services - suteikti paslaugų rinkinį, (prisijungimams prie interneto pvz: DHCP it t.t.)
6 įkelti vartotojo programas - įkelia programas į atmintį ir paleidžia vykdyti

OS būna bendrosios paskirties ir realaus laiko
Bendrosios paskirties - (Windows, Linux, iOS, Android) visos operacijos statomos į eilę
Realaus laiko sistemos - QNX (naudojama automobiliuose, lėktuvuose, medicinoje) reaguoja iš karto, per tam tikrą nustatytą laiką
